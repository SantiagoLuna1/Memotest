# Memotest Game

**Memotest** es un juego de memoria donde los jugadores deben hacer coincidir cuadros de colores. El objetivo es encontrar todos los pares de cuadros iguales en el menor número de intentos.

## Descripción

Este proyecto está hecho con HTML, CSS y JavaScript. El juego genera cuadros con colores aleatorios que los jugadores deben ir destapando hasta que encuentren todos los pares. El juego termina cuando todos los cuadros han sido encontrados.
